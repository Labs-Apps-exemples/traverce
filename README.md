# traverce

> COVID compliant travel

## Contribute

You can submit your contribution to improve the experience of **traverce.net** by submitting your pull requests.

## Install

**Requirement**:

- [📗 NodeJS 10+](https://nodejs.org/en/)
- [🐈 Yarn](https://yarnpkg.com/lang/en/)

```bash
$ yarn
```

## Run

```bash
$ yarn dev
```
